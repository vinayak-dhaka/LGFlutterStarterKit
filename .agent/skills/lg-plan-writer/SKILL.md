# Liquid Galaxy Flutter Plan Writer

## Purpose
Create a structured implementation plan for a Flutter Liquid Galaxy application.

## Responsibilities
- Break project into small tasks
- Ensure Flutter architecture separation
- Include LG connection setup early
- Plan UI, services, API usage

## Output Format
1. Project structure
2. Services to implement
3. Screens/widgets needed
4. External APIs
5. Testing steps

## Guardrails
- Avoid large single-step plans
- Ensure starter kit structure is respected
