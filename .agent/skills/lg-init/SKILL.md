# Liquid Galaxy Flutter Init

## Purpose
Initialize a new Liquid Galaxy Flutter application using the LG Flutter Starter Kit.

## Responsibilities
- Verify repository structure follows starter kit layout
- Ensure LGConnectionService and LGCommandService exist
- Confirm environment configuration (IP, username, password)
- Validate assets folder and demo.kml availability

## Steps
1. Check Flutter project builds successfully.
2. Confirm SSH connection service is functional.
3. Verify LG command execution works.
4. Confirm example screen loads and connects.

## Guardrails
- Do NOT generate a full app immediately.
- Ensure starter kit integrity first.
- Follow separation of concerns.
